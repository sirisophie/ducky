# Ducky

Mobile-first babysitter guide for Ducky.

## Run Locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Notes

- The app is currently a static shell with starter content.
- Gallery uploads are saved to the current browser with IndexedDB.
- To let multiple friends upload photos that you can see from your own device, the gallery will need shared storage or a small backend.
